# BarkFart

Plays a bark and a fart sound effect when throwing basketballs and when they hit surfaces.

<details>
  <summary><b>Installation</b></summary>
	
1. **Download** the latest release **[here](https://github.com/iiDk-the-actual/BarkFart/releases/latest)**
2. **Drag & Drop** `BarkFart.dll` into your plugins folder  
3. **Launch** Rec Room and enjoy!

**From Source Code (for developers!)**

1. Download the source code **[here](https://github.com/iiDk-the-actual/BarkFart/releases/latest)**
2. Edit `Directory.Build.props` and update `<GamePath>` to your Rec Room installation path (e.g. `C:\Program Files\Rec Room`)
3. Build the project with `Ctrl + Shift + B`<Br>
✅ The DLL will automatically go into your game's plugins folder

</details>
